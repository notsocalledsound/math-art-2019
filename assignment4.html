assignment4 : sinner

Today we reviewed what we know about for loops and drawing shapes and added a big new javascripttool: arrays.

We also learned about sine functions, how they can be used to create periodic shapes and also sounds.

Over the next few classes we will be putting all of these pieces together to start creating music visualizations.

For today's assignment, we will keep it simple.

Be sure you have read about "procedural shapes in p5" from today's syllabus.

Make a grid of procedural shapes using for loops and color arrays, as we did in class

You can follow my example below.

Also, go back and make sure you have successfully posted code to every assignment page thus far this semester.

You don't have to go all out on every assignment but you need to finish each one to pass the class!

Finally, think about how you would animate your grid of shapes and experiment with various techniques you think up.

You might try using a sine wave to make them oscillate, as we did in class.

But what if you want to make each one move independently, or at different speeds, or bounce off each other?

How can we do that?

Next class we'll talk about the best ways to go about doing that!

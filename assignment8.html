assignment 8: fun with fourier

Today we first reviewed drawing circles and simple waveforms using sin() and cos().

Then we looked at one of the most useful algorithms of all time, the fourier transform.

We learned how to analyze audio using an FFT amnd also learned how to draw anythinfg we want using waveforms!

For this assignment you should use an FFT to analyze a sound and then use your creativity to draw the output to the screen

Also, post the code we wrote in class as we explored the fourier series.

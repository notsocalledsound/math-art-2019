assignment 6: javascript objects and sound synthesis

Today we continued working with sounds in p5, reviewed map() and constrain() and explored this extremely useful idea called constructor functions.

For this assignment I want you to create a class of constructor function called a SoundBubble.

When SoundBubbles collide with each other, the bounce off each other and sounds are produced!

You can use my example code to get started.

assignment 5: fibs

Today we began working with sounds in p5. Oscillators, sound files, getting amplitude levels and drawing those to the screen.

We also touched on Javascript objects and learned about the map() function for converting amplitude values into more usable values.

For this assignment I want you to visualize some sounds by making a waveform, using these tools and other tools (drawing with vertexes in loops) that we've already learned in class.
